== tile-conflate

=== Description

Conflates tiles input1 and input2 and generates output.

* input1 - Input (e.g. .osm file).
* input2 - Input (e.g. .osm file).
* output - Output file (e.g. .osm file).

=== Usage

--------------------------------------
tile-conflate (input1) (input2) (output)
--------------------------------------
