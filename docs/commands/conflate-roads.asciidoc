== conflate-roads

=== Description
 This will be deprecated in the future. conflate is preferred. Conflates two input sources into one output. Only roads are conflated.
 
 * +input1+ - First input.
 * +input2+ - Second input.
 * +output+ - The output path.

=== Usage
--------------------------------------
conflate-roads (input1) (input2) (output)
--------------------------------------
